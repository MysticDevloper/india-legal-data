📝 How to File a Complaint

1

Theft / Burglary

1. Visit state police portal

2. File FIR under "Online FIR"

3. Get FIR copy for insurance

File Now →
2

Cyber Fraud / Scam

1. Call bank immediately to block

2. File at cybercrime.gov.in

3. Save acknowledgment number

File Now →
3

Domestic Violence

1. Call 181 Women Helpline

2. File complaint at police

3. Apply for free lawyer at NALSA

Get Free Lawyer →
4

Police Not Taking Action

1. Wait 7 days

2. File complaint to SP

3. Escalate at pgportal.gov.in

File Grievance →
